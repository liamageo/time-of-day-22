Greeting goes here

This page demonstrates how to use JavaScript to change a webpage's greeting, background color, and images, according to the time of day. By observing the code in my repository, you will find a breakdown of how it was possible to alter these aspects of the webpage. Many aspects of the code are binded by a specific id, in order for the code to successfully work.

In order to change my greeting to fit the time of the day, I used an id to bind my greeting header with name of myGreeting and JS's getDate and getHours method. This allowed me to choose specific hours of the day, that I want the greeting to change. For example, if it's 8:00 pm, the greeting on the web page will appear as, "Good Evening!". The background color and images are also of the same concept. It's a string of different variables like the times, colors, and images, so they each change accordingly and are also binded to each statement/method. In the code, viewers can find the if, else, and else if statements, which have allowed me to create that string. Again, the main parts of this webpage that are demonstrated and within the code is the day, hours, greeting, background color, and images.
